sega2 css framework

О sega2

Sega2 — небольшая css-библиотека. Базовый элемент — компонент, верстается тегом или классом, стили задаются в components.less. Для сброса оформления по-умолчанию используется sanitize.css.

Вариации элемента задаются модификаторами (класс с префиксом «-»).

Базовые модификаторы:
  • Размера: -small, -mid, -big, -extra
  • Состояния: -disabled, -active, -selected, -open, -collapsed, -hide, -visible
  • Положения: -fixed
  • Перекрытия: -ellipsis, -overflow-hidden
  • Приоритета: -misc
  • Колонки: c1, c2, c3,…, -full
При создании кастомного модификатора, например, для -fixed используется след. порядок: -fixed-header.

Зависимости:

  • Sanitize.css
  • PT-Sans
  • Material design icons
  • ios7switch

Библиотека построена на less-файлах:

  • less/colors.less — цветовая гамма
  • less/grid.less — сетка, отступы
  • less/typographic.less — заголовки и прочее
  • less/components.less — таблицы, формы, хидер, футер, кнопки и т. д.

Установка

bower install sega2 Подключите на своей странице sega2.css <link rel="stylesheet" href="bower_components/sega2/sega2.css" /> или импортируйте себе файл sega2.less, скорее всего придётся в файле sega2.less поправить пути импорта css-файлов. @import "bower_components/sega2/less/sega2.less";
